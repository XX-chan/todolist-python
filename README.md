# Flask-Todolist 
## 项目介绍
Todolist 是一个简单的待办事项网络应用程序，它可以实现登录、应用程序接口和交互式用户界面。

## 技术栈
- Backend：Python(Flask)
- Database: SQLite
- Forntend: HTML + JavaScript

## 功能
- 添加任务
- 删除任务
- 标记完成
- 查看任务列表

## 使用某种技术的原因
使用Python语言：此项目是我的第一个全栈项目，目的是为了能快速了解和掌握实现全栈所需掌握的知识。

该项目使用了：JavaScript、Flask、session、实现了登录功能、API（fetch）、前后端分离（未实现完全分离）、JSON等。
